minify化する

5：　[_minifyset/」をブラウザで叩いてminify化したファイルを作成します。
http://localhost/コンテンツ・ルート/_minifyset/

6：　minify化されたファイルが作成されているか確認します。

「top.min.css」と「top.min.js」が作成されて、ディレクトリ構成は以下のようになります。 
コンテンツ・ルート/
	├ _minifyset/
	├ assets/
	│	├ css/
	│	|	├ top.css
	│	|	└ top.min.css * <-- 作成されたminifyファイル
	│	├ js/
	│	|	├ top.js
	│	|	└ top.min.js * <-- 作成されたminifyファイル
	│	└ minifyset.config.js *
	└ common/
	index.html
	minifyset.loader.js *
7：　ローカル環境時の読み込みを確認する。
注意：　ローカル環境のホスト名は初期値で「localhost」に設定されています。違う場合は「minifyset.loader.js」の変数「localhost」の値を変更してください。

http://localhost/コンテンツ・ルート/
--> ローカル環境の時、下記のようにファイルを読み込みます。 

<link href="./common/css/common.css" rel="stylesheet" type="text/css">
<link href="./assets/css/top.css" rel="stylesheet" type="text/css">
<link href="./common/lib/bootstrap-4.5.3.css" rel="stylesheet" type="text/css">
<link href="./common/lib/highlightjs-railscasts.min.css" rel="stylesheet" type="text/css">
<script type="text/javascript" src="./common/lib/jquery-3.6.0.js"></script>
<script type="text/javascript" src="./common/lib/bootstrap-4.5.3.js"></script>
<script type="text/javascript" src="./common/lib/highlight.min.js"></script>
<script type="text/javascript" src="./common/lib/jquery.clone_blocks.js"></script>
<script type="text/javascript" src="./common/lib/jquery.inview.min.js"></script>
<script type="text/javascript" src="assets/js/top.js"></script>
8：　公開環境時の読み込みを確認する
ローカル環境で公開時の確認をする：　「minifyset.loader.js」にある配列「_localhost」の値から「localhost」を削除する。
var _localhost = ["localhost", "....."];
　　　　↓
var _localhost = ["....."];

http://公開環境ドメイン/コンテンツ・ルート/
--> 公開環境の時、minify化されたファイルを読み込みます。 

<link href="./assets/css/top.min.css" rel="stylesheet" type="text/css">
<script type="text/javascript" src="./assets/js/top.min.js"></script>
